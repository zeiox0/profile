# 🎯 موقع شخصي احترافي - Abdallah Official

موقع شخصي حديث وسريع مع لوحة تحكم متقدمة لإدارة الفيديو والصوت والبيانات الشخصية.

---

## ✨ الميزات الرئيسية

### 🎬 إدارة الفيديو والصوت
- **ضغط تلقائي**: الفيديوهات والأصوات يتم ضغطها تلقائياً قبل الرفع للحفاظ على الجودة وتقليل الحجم
- **دعم الروابط المباشرة**: يمكنك إدراج رابط مباشر أو رفع ملف من جهازك
- **رفع آمن**: جميع الملفات يتم رفعها إلى Supabase بشكل آمن

### 📱 عرض متجاوب (Responsive Design)
- **دعم كامل للهواتف**: تصميم يتكيف مع جميع أحجام الشاشات
- **تجربة سلسة**: نفس الجودة والأداء على الكمبيوتر والهاتف
- **أدوات تحكم سهلة**: أزرار وأشرطة تمرير للتحكم في الفيديو والصوت

### 🎮 أدوات التحكم المتقدمة
- **تشغيل/إيقاف الفيديو**: التحكم الكامل في تشغيل الفيديو
- **التحكم في السطوع**: ضبط سطوع الفيديو حسب الرغبة
- **التحكم في الصوت**: مستوى الصوت والتشغيل/الإيقاف
- **واجهة سهلة**: جميع الأدوات في شريط واحد في أسفل الصفحة

### 🔒 الأمان والخصوصية
- **تسجيل دخول آمن**: حماية صفحة الإدارة برسالة بريد إلكترونية وكلمة مرور
- **Firebase Authentication**: استخدام Firebase للمصادقة الآمنة
- **Supabase Storage**: تخزين آمن للملفات

### 🔍 تحسين محركات البحث (SEO)
- **تحديد النطاق**: النطاق محدد على `zeiox0.github.io` في جوجل Search Console
- **Sitemap**: ملف sitemap.xml لتحسين الفهرسة
- **Robots.txt**: ملف robots.txt لتوجيه محركات البحث
- **Meta Tags**: علامات وصفية محسّنة

## 🚀 البدء السريع

### 1. التثبيت
```bash
# استنساخ المشروع
git clone <repository-url>
cd profile-main

# لا توجد متطلبات تثبيت إضافية - الموقع يعمل مباشرة
```

### 2. الإعدادات
- تحديث `config.js` بمفاتيح Firebase الخاصة بك
- تحديث `admin.js` بمفاتيح Supabase الخاصة بك
- تحديث البريد الإلكتروني المصرح به في `admin.js`

### 3. الرفع
```bash
# رفع الملفات إلى GitHub Pages أو أي خادم ويب
# تأكد من أن النطاق محدد على zeiox0.github.io
```

## 🎯 كيفية الاستخدام

### إضافة فيديو أو صوت

1. اذهب إلى صفحة الإدارة (`admin.html`)
2. سجل الدخول بالبريد الإلكترونية وكلمة المرور
3. اختر الفيديو أو الصوت من جهازك
4. اضغط على "تحديث الفيديو" أو "تحديث الصوت"
5. انتظر حتى ينتهي الضغط والرفع

### تعديل البيانات الشخصية

1. في صفحة الإدارة، أدخل اسمك والوصف
2. اضغط على "حفظ البيانات"
3. سيتم تحديث الموقع الرئيسي تلقائياً

### التحكم في الفيديو والصوت

1. على الصفحة الرئيسية، ستجد أدوات التحكم في أسفل الشاشة
2. استخدم الأزرار للتشغيل/الإيقاف
3. استخدم أشرطة التمرير للتحكم في مستوى الصوت والسطوع

## ⚙️ إعدادات الضغط

### الفيديو
- **الجودة (CRF)**: 23 (توازن بين الجودة والحجم)
- **معدل البت**: 2500k (جودة متوسطة)
- **الترميز**: H.264 (متوافق مع جميع الأجهزة)

### الصوت
- **معدل البت**: 192k (جودة عالية)
- **الجودة**: VBR 4 (متغير)
- **الترميز**: MP3 (متوافق مع جميع الأجهزة)

## 🔧 التعديلات والتخصيص

### تغيير ألوان الموقع
عدّل ملف `style.css` وغيّر قيم الألوان:
```css
#user-name { color: #fff; } /* اللون الأبيض */
.control-btn { background: #007bff; } /* الأزرق */
```

### تغيير إعدادات الضغط
عدّل ملف `media-compressor.js`:
```javascript
quality: 23, // غيّر هذا الرقم (0-51)
bitrate: '2500k', // غيّر معدل البت
```

### إضافة روابط السوشيال ميديا
عدّل ملف `index.html` وأضف الروابط في قسم `social-links`

## 🌐 النطاق والبحث في جوجل

الموقع مُعدّ للعمل فقط على النطاق `zeiox0.github.io`:

- **robots.txt**: يحدد النطاق الأساسي
- **.htaccess**: يعيد توجيه جميع الطلبات إلى النطاق الصحيح
- **sitemap.xml**: يساعد جوجل في فهرسة الموقع

لتحسين الفهرسة:
1. أضف الموقع إلى Google Search Console
2. أرسل sitemap.xml
3. تحقق من ملف robots.txt

## 🐛 استكشاف الأخطاء

### الفيديو لا يظهر
- تحقق من أن رابط الفيديو صحيح
- تأكد من أن الملف بصيغة MP4
- تحقق من إعدادات Supabase Policies

### الصوت لا يشتغل
- تحقق من أن رابط الصوت صحيح
- تأكد من أن الملف بصيغة MP3
- تحقق من مستوى الصوت في الأدوات

### الضغط بطيء جداً
- هذا طبيعي للملفات الكبيرة
- استخدم ملفات أصغر حجماً
- تحقق من سرعة الإنترنت

### الرفع يفشل
- تحقق من إعدادات Supabase
- تأكد من أن اسم المخزن صحيح
- تحقق من Policies في Supabase
